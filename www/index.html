Nuevas Tecnologias

¿Que son las NTD?

En nuevas tecnologias de desarrollo, o NTD como sus siglas lo indican, aprenderemos los modelos de desarrollo de software para crear aplicaciones modernas con gran capacidad de escalabilidad a la mayoria de dispositivos tecnologicos como las computadoras, los celulares y tablets. Aprenderemos gracias a los modelos de desarrollo como MVC, sobre el funcionamiento de los servidores, y el diseño y desarrollo de aplicaciones con HTML 5, CSS 3, JavaScript, Java e incluso podriamos ver Phyton. Tambien vamos a ver sobre los frameworks de trabajo como Angular, JS, Node JS. Gracias a dichas bases para aprender y entender otros lenguajes y modelos de trabajo como puede ser Ruby y su framework Rails.

Temas de la clase

Modelo Vista Controlador

Es un patron que pertenece al la familia componente-conector, separa la lógica de negocio de la interfaz de usuario. Facilita la evolución por separado de ambos aspectos, incrementando de esta forma la reutilización y flexibilidad.

Servidores

Es el componente fisico o virtual capaz de realizar el desplique de la aplicación, es donde se encuentra la información de la base de datos y las vistas a las cuales el usuario queire aceder por peticiones.

Arquitectura JPA

Es una API que proporciona un mecanismo para gestionar la persistencia y la correlación y funciones relacionales de objetos.

Controlador

El controlador se encarga de realizar la unión de la información con las vista de esta al usuario final, en este trancurso de solicitudes, veremos el CRUD "crear leer actualizar y borrar", peticiones entre otras cosas mas como la utilización de frameworks.

Vista

Empieza la parte de diseño y desarrollo front-end, todo lo que se ejecuta en la computadora del usuario fuera del servidor, aprenderemos el etiquetado de HTML5 y con esto podras utilizar preprocesadores como HAML JADE entre otros. para dar estilos CSS aprenderemos el uso de preprocesadores como SASS, y la iteracción con el usuaro con el uso de JavaScript.

Django

Es un potente framework de desarrollo Web con el que podemos utilzar el leguanje Phyton para el desarrollo de aplicaciones, agregando velocidad al desarrollo y seguridad.

Node JS

¿ JavaScript desde el servidor?, Node JS es JavaScript orientado a eventos asíncronos, esta diseñado para construir aplicaciones en red escalables.

Formas de trabajo

Hasta el momento estamos acostumbrados a dividir el trabajo y despues juntarlo al final de la antrega del proyecto. Ahora con herramientas como Git aprenderemos a trabajar en linea y de forma simultanea en el mismo proyecto y uni todas las parte al tiempo del desaarrollo. Inclusive podemos trabajar al tiempo en la misma parte del desarrollo.
